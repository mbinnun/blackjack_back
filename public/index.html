The REST API of Blackjack Game

Author: Moshe Binnun

BlackjackGame Rest API

Backend API files of blackjack game.

To install

git clone https://.../blackjack-app-back
cd blackjack-app-back
cp .env.example .env

- fill in all the parameters in the .env file. Tune them to your needs

- on your mongodb server, create a db named GAMES and a collection name TblGames

npm install

Start the API server

npm start

The Restful APIs collection

[GET]: /api/games/
[Description] Fetch the games list

[GET]: /api/games/{:id}
[Description] Fetch the details of a game by id

[POST]: /api/games/
[Description] Register a new game

[PUT]: /api/games/restart/{:id}
[Params] id
[Description] Restart a game to the initial state

[PUT]: /api/games/bet/{:id}
[Params] id, bet
[Description] Place a bet in a game
[Rem] Bet should be an integer number

[PUT]: /api/games/take/{:id}
[Params] id
[Description] Player takes a card in a game

[PUT]: /api/games/expose/{:id}
[Params] id
[Description] Expose all the cards in a game (including dealer cards)
